# Bulk File Renameing script

## Overview
This script can be used to rename all the files in the same folder/directory in a systematic way, necessary for renaming bulk files.

## Features
- Renames all the files in the desired directory/folder.
- Ignores all the sub-folders in the folder.
- Prints all the previous file name with the new file name when changed.

## Setup
1. Clone the repository
<pre>git clone https://github.com/Arman-Ali-Dafadar/renaming-script.git
cd renaming-script</pre>

2. Run the app using:
<pre>
python3 app.py folder_path
</pre>
>Here folder_path should be replaced by the actual or relative path of the folder/directory

## Final Results
All the files in the desired folder will be renamed and all the previous filename with the current filename will be printed in the terminal.
